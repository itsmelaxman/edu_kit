# Edu Kit

Edu Kit is a flutter package for building education-based mobile apps. It offers a pre-built education app kit with screens, widgets, and components to create engaging learning experiences.

## Features [Widgets]

- **Assignment**
- **Attendance**
- **Bus [Bus Route]**
- **Calendar**
- **Chat**
- **Complaint**
- **Digital Card**
- **E-Payment**
- **E-Learning**
- **Exam Feedback**
- **Exam Result**
- **Exam Routine**
- **Leave**
- **Lesson Plan**
- **Library**
- **Notice [Timeline]**
- **Notification**
- **Online Exam**
- **Profile**
- **Timetable**
