## Features [Widgets]

- **Assignment**
- **Attendance**
- **Bus [Bus Route]**
- **Calendar**
- **Chat**
- **Complaint**
- **Digital Card**
- **E-Payment**
- **E-Learning**
- **Exam Feedback**
- **Exam Result**
- **Exam Routine**
- **Leave**
- **Lesson Plan**
- **Library**
- **Notice [Timeline]**
- **Notification**
- **Online Exam**
- **Profile**
- **Timetable**
